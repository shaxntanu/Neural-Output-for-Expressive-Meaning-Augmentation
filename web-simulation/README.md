# NOEMA Signal Sandbox

Interactive biosignal visualization platform for the NOEMA research project.

## What This Does

This web app simulates brain signals (EEG-like patterns) in real-time based on different mental states and intentions. You can adjust parameters like cognitive intent (Yes/No/Help), emotional state (Calm/Stressed/Focused), and see how synthetic biosignals change across different brain regions.

Use it to prototype and visualize what brain-computer interface signals might look like under different scenarios.

## ⚠️ Disclaimer

All signals are **synthetic and algorithmically generated**. This is NOT real brain data or medical equipment. It's a visualization tool for research planning and demonstration.

## Features

- Real-time multi-channel signal visualization (Frontal, Motor, Temporal, Physiological)
- Control cognitive intent, emotional state, eye state, noise, and signal strength
- Live analytics: stress index, focus level, signal stability, intent confidence
- Interactive brain region visualization
- Scenario presets (Meditation, Exam Stress, Silent Yes, Emergency Help)
- Export data as JSON or CSV
